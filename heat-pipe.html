Portfolio Details

This is design, validation and assembly of a racing go-kart.

Project Overview

This project presents a CFD-based evaluation of a novel heat pipe design intended for high-temperature fusion environments. The simulation was performed using a 2D axisymmetric model in ANSYS Fluent, as part of a Master’s-level dissertation.

Working Principle

This heat pipe uses a liquid metal wick that evaporates at the hot end and condenses at the cooler end, recirculating via capillary action. This setup enables passive thermal regulation under fusion-relevant conditions.

2D Geometry Simplification

The original 3D model was idealized into a 2D axisymmetric domain to reduce computational load while preserving thermal-fluid behavior. Heat flux zones were created using geometric imprints.

Mesh Refinement & Quality

Refined mesh was applied to regions of steep gradients including the evaporator and condenser ends. Higher-order 2D quadrilateral elements were used to maintain solution stability and convergence.

Mesh Metrics

Mesh metrics such as skewness and aspect ratio were assessed to maintain high quality throughout the simulation. These helped control divergence during iterations.

Mesh Statistics

The mesh convergence graph shows how the number of elements and element size affects computational efficiency. This optimization reduced solve time significantly.

Temperature Distribution

Although full phase-coupled multiphase modeling did not converge due to interface instability, the temperature contours closely matched benchmark data from COMSOL models at TU/e, validating the approach.

Cross-Validation

The high-resolution zoom confirms that the simplified Fluent model reproduces realistic thermal gradients despite lacking explicit liquid-vapor interface modeling.

Conclusion

This study demonstrates the effectiveness of a reduced-order CFD model in predicting fusion-relevant thermal transport. Future development can incorporate phase change via VOF or OpenFOAM to further expand fidelity.
