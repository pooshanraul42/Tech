Fusion-Relevant Heat Pipe Simulation

This project presents a CFD-based evaluation of a novel heat pipe design intended for high-temperature fusion environments. The simulation was performed using a 2D axisymmetric model in ANSYS Fluent, as part of a Master’s-level dissertation.

2D Geometry Simplification

The original 3D model was simplified to a 2D axisymmetric domain to reduce computational load while preserving thermal-fluid behavior. Heat flux and temperature-dependent properties were defined via imprinting techniques.

Internal Capillary Structure

The model includes a simplified capillary wick structure. Phase change behavior was intended via fluid-fluid interface modeling, though convergence was limited due to solver constraints.

Mesh Refinement & Quality

Local refinement was applied in the heat input and condensation regions. Element size was optimized for resolving gradients without excessive cell count.

Mesh quality and size distribution overview for convergence analysis.

Temperature Contours

Although the full fluid-fluid interface did not converge, the resulting temperature distribution closely matched COMSOL-based results from TU/e researchers, supporting the accuracy of the simplified model.

Conclusion

This study demonstrates the feasibility of using a reduced-order 2D thermal-fluid model for evaluating fusion-compatible heat pipe concepts. Further work may integrate phase change and surface tension modeling to simulate condensation and evaporation cycles in detail.
