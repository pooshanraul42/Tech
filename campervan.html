CAD-to-CNC Engineering of Campervan Furniture Kits

Flat-pack kit conversion using SolidWorks and VCarve Pro. Focused on parametric modeling, joinery optimization, and toolpath generation for batch manufacturing.

Engineered flat-pack campervan kits from legacy DXFs by rebuilding full parametric 3D assemblies. Used SolidWorks for modular redesign and joinery setup, and VCarve Pro for toolpath generation. Focused on improving manufacturability, material efficiency, and repeatable installation.

3D CAD Reconstruction from 2D Kits

Reconstructed CNC-ready 3D assemblies from raw 2D DXF files. Fully constrained part models allowed toolpath alignment, modular edits, and joinery placement while enabling layout-specific tweaks with minimal design time.

  • Rebuilt parametric assemblies with joinery references and slot alignment
  • Established uniform design constraints across all variants
  • Streamlined the CAD-to-CAM flow using consistent export templates

Each layout was restructured to support efficient installation, modular upgrades, and minimal operator dependency. Joinery placement, part ID tags, and fastener locations were engineered in CAD for precision routing and long-term durability.

  • Cam-lock systems modeled and placed using CNC-ready reference features
  • Fastener zones optimized to preserve ply structure and assembly repeatability
  • All internal openings routed with tool clearance logic for zero hand-finishing

Production-Ready Layout with DFM Features

Reconfigured the kits to maximize board utilization, simplify access zones, and improve modularity. Focus was placed on reducing cut complexity, isolating utility features, and improving downstream manufacturability.

  • Nesting strategy tuned for standard plywood sheet sizes (2440Ã—1220 mm)
  • Power and utility zones isolated for safety and simplified access
  • Feature packages defined to enable scalable product tiers

Component geometry was updated to include appliance cavities, wire routing access, and multi-material integration for worktops. Structural loads and bracket mounting zones were accounted for in CAD.

  • Appliance locations pre-machined into panel assemblies
  • Internal routing zones defined for cables, tanks, and control hardware
  • Modular framing allowed component upgrades without redesign

CNC Tooling Strategy for Clean Finishing

Compression tooling was used to eliminate bit swapping between upcut and downcut passes, maintaining surface quality across both faces. Separate feed and spindle settings were tuned for birch plywood and bamboo to prevent edge burning and delamination.

  • Compression bits used to minimize tear-out on both faces
  • Cut depth, RPM, and feed rates calibrated by material type
  • Toolpath optimized to avoid tool wear and manual finishing

All components were processed in VCarve Pro using reusable toolpath templates for profiling, pocketing, and part ID engraving. Scrap protective plywood from packaging was used for real-scale test cuts, enabling fast validation before final machining.

  • Reusable VCarve templates ensured CNC consistency
  • Fitment trials performed on recycled plywood blanks
  • Embedded engravings improved part identification and orientation

Machining Outcome and Install Feedback

The CNC-manufactured kits were assembled without any rework. All components fit with high dimensional accuracy, and multiple batches were produced using the same toolpath set with consistent results.

  • ~90% reduction in post-processing effort (sanding, trimming)
  • Toolpath workflow reused across different van models
  • Field installs confirmed tight fit and minimal in-transit rattling
