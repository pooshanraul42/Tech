3D Piping & Equipment Layout – Solvent Recovery Plant

Designed using SolidWorks, this project illustrates equipment placement, piping routes, and mechanical integration for an industrial solvent recovery process.

The project involved developing a modular equipment layout for a solvent recovery plant. Emphasis was placed on accessibility, equipment spacing, and clean routing of process pipelines. Core design principles included adherence to space management, equipment orientation, and feasibility for manufacturing, installation and maintenance. All models were created using SolidWorks.

Overall Plant Layout

This overview shows all major units in position — boiler, reactor, condensers, storage tanks, and interconnecting pipelines—highlighting clearances and logical placement for ease of access and operation.

Dimetric Piping Layout

Dimetric projection provides clarity on 3D routing with proper elevation levels. This helped validate pipe run efficiency and supported isolation checks and support placements.

Boiler and Cooling Zone

Focused render showing thermal equipment placement. Considerations for ventilation, structural support, and servicing zones were incorporated into the design stage.

Detailed Condenser Models

These vessels were modeled with internal specifications based on manufacturing standards. The design includes shell thicknesses, nozzle placements, and structural elements like tubesheets and baffles.

This demonstrates that the assemblies are not just visual shells, but represent manufacturing-ready components suitable for fabrication and assembly-level documentation.

P&ID Reference

This snapshot represents the reference diagram used during CAD modeling. It informed line numbering, instrument connections, material of construction, and flow sequencing for the plant.
